# PRD

プロダクトの目的と MVP 範囲です。

## Background

在校生が OB/OG の就職先や選考体験を知る機会は、先生や知人経由に偏りやすいです。

このサイトは、先輩の就活知見を学校内で見つけやすくするためのナレッジベースです。

## Users

- 在校生: 志望企業や同じ学科の先輩を探す
- OB/OG: 自分の経験を後輩に共有する
- 管理者: 利用者と公開情報を管理する

## MVP Features

### Login

- Google OAuth でログインする
- 許可されたドメイン、管理者メール、引き継ぎ Gmail を扱う

### Initial Setup

- 名前、学籍番号、入学年度、学科を登録する
- 学科から修業年数を決める

### Alumni List

- OB/OG をカードで表示する
- 学科、卒業年度、企業名で絞り込む
- 表示件数とページを切り替える

### Alumni Detail

- プロフィールを表示する
- 企業別の選考体験を表示する
- SNS 連絡導線を表示する

### Account

- 基本情報を更新する
- 公開プロフィールを更新する
- アバター画像をアップロードする
- 引き継ぎ Gmail を登録する
- アカウントを削除する

## Profile Fields

- 表示名
- 卒業年度
- 学科
- 内定先、就職先
- 備考
- X URL
- Instagram URL
- スキル
- ポートフォリオ URL
- 学チカ
- 役に立った授業
- 企業別の選考体験

## Selection Experience

企業ごとに以下を登録します。

- エントリーのきっかけ
- 全体 Tips
- 選考ステップ
- 実施形式
- 面接官人数
- 所要時間
- 聞かれた質問
- 雰囲気
- 準備してよかったこと

## Success Metrics

- 公開プロフィール数
- 一覧閲覧数
- 詳細閲覧数
- SNS 連絡導線の利用数
